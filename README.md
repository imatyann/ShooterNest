# ShooterNest(シューティネスト) -MVP v0.1.0

9*9のグリッド上で、3体の駒をクリック操作で動かしてリアルタイムで攻撃するシューティングゲーム

## 1.遊び方(MVP)
- 左クリック：駒を選択,上下左右の空きマスへ移動
- 右クリック：選択中の駒が上方向に弾を発射
- R：リトライ／Esc：終了
- 勝利：弾が敵にあたる／敗北：敵味方の存在マスが重なる
